# Итерация 29 — Реализация поиска трека (/search)

## Задача

Добавить команду `/search` для поиска треков.

## Поведение команды

1. Пользователь вводит `/search`
2. Бот направляет текст инструкции для поиска (например, `Введите информацию для поиска в формате Артист - Песня. Например, Полина Гагарина - Shallow`)
3. Пользователь вводит ответ (например, `Полина Гагарина - Shallow`)
4. Бот выполняет поиск:
   - Сначала на локальном хранилище (в директории `{TRACKS_ROOT_DIR}`)
   - Поиск с нормализацией имени файла (без учёта регистра и знаков препинания)
   - Фильтрация: исключаются файлы с `(Instrumental)` и `(Vocal)`
   - Если не найдено локально — выполняет поиск на Яндекс Музыке по запросу пользователя
5. Возвращает список найденных треков с возможностью выбора для дальнейшей обработки

## Реализация

### 1. Регистрация команды

- Добавлена команда `/search` в список команд бота

### 2. Поиск по локальному хранилищу

- Сканирование директории `{TRACKS_ROOT_DIR}` на 1 уровень вглубь
- Фильтрация по расширениям: `.mp3`, `.flac`, `.mp4`, `.avi`
- Исключение файлов с `(Instrumental)` и `(Vocal)`
- Строгий поиск по одновременному наличию artist и title
- Проверка наличия `state.json` в папке, чтение `track_source`
- Сортировка по времени создания (убывание), топ-5

### 3. Поиск на Яндекс Музыке

- Реализован метод поиска на Яндекс Музыке через API

### 4. Отображение результатов

- Горизонтальные inline кнопки 1–5
- Кнопка "Я" для принудительного поиска на Яндекс Музыке

### 5. Обработка выбора

- **Локальный трек с `state.json`**: продолжение с последнего шага или перезапуск `RENDER_VIDEO`
- **Локальный трек без `state.json`**: полный пайплайн с копированием файла
- **Яндекс Музыка**: скачивание и полный пайплайн
- **При `COMPLETED` с готовым `output_file`**: отправка ссылки без перезапуска

## Дополнительно добавлено

- Логирование запросов пользователя при поиске треков
- Логирование результатов поиска (количество локальных и яндекс треков)

## Проверка

Команда `/search` запускает процесс поиска с корректной фильтрацией и нормализацией.
