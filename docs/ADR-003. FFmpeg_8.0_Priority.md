# ADR-003: Приоритет FFmpeg 8.0 над стабильностью LTS

## Статус

Принято (пересмотр ADR-002 и анализа итерации 53)

## Контекст

После анализа итерации 53 ([`docs/ffmpeg-comparison.md`](docs/ffmpeg-comparison.md)) было принято решение остаться на FFmpeg 6.1.1 из Ubuntu 24.04 LTS.

**Новое условие:** FFmpeg 8.0 является приоритетом для проекта. Необходимо пересмотреть решение и найти оптимальный путь получения FFmpeg 8.0.

## Проблема

Как получить FFmpeg 8.0 с минимальными компромиссами в стабильности и сложности инфраструктуры?

## Варианты решения

### Вариант A: Ubuntu 25.04 (Plucky Puffin) с FFmpeg из репозитория

Ubuntu 25.04 содержит FFmpeg 7.1.x (не 8.0).

| Критерий | Оценка |
|----------|--------|
| Версия FFmpeg | 7.1.x (не 8.0) ❌ |
| Поддержка | 9 месяцев (до января 2027) ⚠️ |
| Стабильность | Interim release, не LTS ⚠️ |
| Сложность | Простая установка из репозитория ✅ |

**Вывод:** Ubuntu 25.04 **не решает задачу** — в ней нет FFmpeg 8.0.

---

### Вариант B: Ubuntu 24.04 LTS + FFmpeg 8.0 через PPA

Установка FFmpeg 8.0 через сторонний PPA в Ubuntu 24.04.

| Критерий | Оценка |
|----------|--------|
| Версия FFmpeg | 8.0 ✅ |
| Поддержка | LTS до 2029 ✅ |
| Стабильность | Зависит от качества PPA ⚠️ |
| Сложность | Средняя (добавление PPA) |

**PPA для FFmpeg:**
- `ppa:jonathonf/ffmpeg-4` — устарел
- `ppa:ubuntuhandbook1/ffmpeg` — может содержать 7.x
- Сборки от Jonathan Temperton (если доступны)

**Риски PPA:**
- Неофициальные сборки
- Возможна задержка обновлений
- Зависимости могут конфликтовать с системными

---

### Вариант C: Ubuntu 24.04 LTS + FFmpeg 8.0 статическая сборка

Скачивание и использование статической сборки FFmpeg 8.0.

| Критерий | Оценка |
|----------|--------|
| Версия FFmpeg | 8.0 ✅ |
| Поддержка | LTS до 2029 ✅ |
| Стабильность | Официальные сборки от johnvansickle.com ✅ |
| Сложность | Низкая (wget + tar) ✅ |
| Размер | +40-60 MB к образу ⚠️ |

**Установка:**
```dockerfile
RUN apt-get update && apt-get install -y wget xz-utils \
    && wget https://johnvansickle.com/ffmpeg/releases/ffmpeg-8.0-amd64-static.tar.xz \
    && tar -xf ffmpeg-8.0-amd64-static.tar.xz \
    && mv ffmpeg-8.0-amd64-static/ffmpeg /usr/local/bin/ \
    && mv ffmpeg-8.0-amd64-static/ffprobe /usr/local/bin/ \
    && rm -rf ffmpeg-8.0-amd64-static ffmpeg-8.0-amd64-static.tar.xz \
    && apt-get remove -y wget xz-utils \
    && rm -rf /var/lib/apt/lists/*
```

**Преимущества:**
- Официальные сборки от доверенного источника
- Нет конфликтов зависимостей
- Простота обновления
- Полный контроль версии

**Недостатки:**
- Увеличение размера образа
- Отдельное обновление FFmpeg

---

### Вариант D: Ubuntu 24.04 LTS + сборка FFmpeg 8.0 из source

Сборка FFmpeg 8.0 из исходников.

| Критерий | Оценка |
|----------|--------|
| Версия FFmpeg | 8.0 ✅ |
| Поддержка | LTS до 2029 ✅ |
| Стабильность | Полный контроль ✅ |
| Сложность | Высокая ❌ |
| Время сборки | 10-20 минут ❌ |

**Вывод:** Слишком сложно и долго для данного проекта.

### Сравнительная таблица

| Вариант | FFmpeg 8.0 | LTS | Стабильность | Сложность | Размер | Рекомендация |
|---------|------------|-----|--------------|-----------|--------|--------------|
| A: Ubuntu 25.04 | ❌ 7.1.x | ❌ 9 мес | ⚠️ Interim | ✅ Низкая | Базовый | ❌ |

## Решение

**Вариант C: Ubuntu 24.04 LTS + FFmpeg 8.0 статическая сборка**

### Обоснование

1. **FFmpeg 8.0 достигнут** — статическая сборка предоставляет нужную версию
2. **Стабильность сохранена** — база Ubuntu 24.04 LTS, проверенные сборки
3. **Простота** — одна команда для установки
4. **Нет конфликтов** — статическая сборка не зависит от системных библиотек
5. **Приемлемый размер** — +40-60 MB для получения FFmpeg 8.0

### Альтернатива: PPA (Вариант B)

Если появится надёжный PPA с FFmpeg 8.0 для Ubuntu 24.04 — можно рассмотреть переход на него для уменьшения размера образа.

## Последствия

### Положительные
- FFmpeg 8.0 с улучшенной производительностью (до 15% на `tune=stillimage`)
- Улучшенное качество AAC кодирования
- Исправления многодорожечного аудио
- Более качественный рендеринг ASS-субтитров

### Отрицательные
- Увеличение размера Docker образа на ~40-60 MB
- Необходимость отдельного обновления FFmpeg
- Зависимость от внешнего источника сборок

## Реализация

Обновлённый [`Dockerfile.ubuntu24`](Dockerfile.ubuntu24) с FFmpeg 8.0 статической сборкой.

```dockerfile
# Статическая сборка FFmpeg 8.0
RUN apt-get update && apt-get install -y --no-install-recommends wget xz-utils \
    && wget -q https://johnvansickle.com/ffmpeg/releases/ffmpeg-8.0-amd64-static.tar.xz \
    && tar -xf ffmpeg-8.0-amd64-static.tar.xz \
    && mv ffmpeg-8.0-amd64-static/ffmpeg /usr/local/bin/ffmpeg \
    && mv ffmpeg-8.0-amd64-static/ffprobe /usr/local/bin/ffprobe \
    && rm -rf ffmpeg-8.0-amd64-static* \
    && ffmpeg -version | head -1 \
    && apt-get purge -y wget xz-utils \
    && apt-get autoremove -y \
    && rm -rf /var/lib/apt/lists/*
```
