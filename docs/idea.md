Необходимо создать бота для автоматического опредления исполнителя для тикетов по задачам технической поддержки

Бот должен обучиться на существующей базе заявок, распределить их по темам, типам и отдел решивший тикет, и в дальнейшем для новых тикетов автоматически предлагать отдел, на который назначать тикет к исполнению.
