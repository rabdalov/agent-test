# Итерация 19 — Корректировка шага GET_LYRICS (пропуск при наличии файла)

## Описание

Модифицировать шаг `GET_LYRICS` таким образом, чтобы перед попыткой загрузки текста песни проверялось наличие файла с текстом в папке трека.

Если существует файл `"source_lyrics_file"` из `state.json`, шаг загрузки пропускается.

## План реализации

1. В начале шага `GET_LYRICS` проверить наличие файла `"source_lyrics_file"` в папке трека.
2. Убедиться, что файл не пустой.
3. Если файл найден — считать его содержимое и использовать как текст песни.
4. Если файл не найден — выполнить обычный поиск через поставщиков.

## Проверка

При наличии текстового файла в папке трека шаг `GET_LYRICS` завершается без обращения к внешним поставщикам.
