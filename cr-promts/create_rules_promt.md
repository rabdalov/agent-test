/create-rule Создай правило (cursor rule) с именем conventions c правилами для разработки кода для передачи их code-ассистенту, который будет генерировать код

Правила должны отражать все главные наши принципы разработки из документа @docs/vision.md и ссылаться на сам документ, не дублируя информацию из него.

Правила должны быть лаконичными, по принципу KISS, не содержать лишнего, только главное влияющее на качество.
