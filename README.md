Телеграм-бот для подготовки караоке-видео.

## Запуск локально

1. Установите зависимости (через `uv`):

```bash
uv sync
```

2. Скопируйте `example.env` в `.env` и заполните значения:

```bash
cp example.env .env
```

3. Экспортируйте переменные окружения (или используйте менеджер окружения по вашему выбору).

4. Запустите бота:

```bash
uv run python -m app.main
```
