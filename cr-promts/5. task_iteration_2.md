### Итерация 2 — формулировка плана

- **Конфигурация хранилища треков**
  - В `Settings` (`config.py`) добавлено поле `tracks_root_dir: Path`, читаемое из `TRACKS_ROOT_DIR` (по умолчанию — `<корень проекта>/tracks`).
  - В `example.env` добавлен блок:
    - `TRACKS_ROOT_DIR=/tracks`.

- **Обработка аудиофайла и формирование `track_name`**
  - В `handlers_karaoke.py`:
    - Для `/start` бот теперь просит прислать аудиофайл-композицию > 1 минуты.
    - Добавлен хендлер для `F.audio`:
      - Скачивает аудиофайл во временную папку `tracks_root_dir/_tmp`.
      - Через `ffprobe` (часть `ffmpeg`) получает:
        - длительность трека;
        - ID3-подобные теги `artist`/`title` (если есть).
      - Формирует `track_name`:
        - если есть теги — `Артист_Название`;
        - если тегов нет — имя файла без пути и расширения;
        - имя нормализуется: пробелы и спецсимволы → `_`, обрезка по краям.
      - Если длительность \< 60 секунд или не удалось её определить:
        - временный файл удаляется;
        - пользователю отправляется сообщение:  
          **«Полученный файл "{track_name}" не является музыкальной композицией (длительность менее 1 минуты или не удалось определить длительность).»**
      - Если длительность ≥ 60 секунд:
        - генерируется `track_id` (`uuid4().hex`);
        - создаётся папка `{TRACKS_ROOT_DIR}\{track_name}\`;
        - файл перемещается в `{TRACKS_ROOT_DIR}\{track_name}\{track_name}.mp3`;
        - пользователю отправляется ответ с:
          - `track_id`,
          - `track_name`,
          - полным путём к сохранённому файлу.

- **Поведение для не-аудио сообщений**
  - Для любых сообщений без аудио (текст и прочее) отдельный хендлер отвечает:  
    **«Полученное сообщение не является музыкальной композицией. Пожалуйста, отправьте аудиофайл длительностью более 1 минуты.»**

### Предлагаю проверить

1. Задать `TRACKS_ROOT_DIR` (например, в `.env` по аналогии с `example.env`).
2. Отправить боту:
   - аудиофайл **\< 1 минуты** — ожидать ответ об отсутствии музыкальной композиции;
   - аудиофайл **\> 1 минуты**:
     - должна появиться папка `{TRACKS_ROOT_DIR}\{track_name}\{track_name}.mp3`;
     - бот вернёт `track_id`, `track_name` и путь к файлу.
3. Отправить текстовое сообщение — увидеть ответ, что сообщение не является музыкальной композицией.
